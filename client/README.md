# Lab 8 React SPA

This folder contains the Lab 8 single-page application built with React, Vite, Ant Design, and React Router.

## Run locally

```bash
cd client
npm install
npm run dev
```

## Build

```bash
cd client
npm run build
```

## What is included

- `Hello` component with a default `Hello World` fallback
- `Goodbye` component with state and mouse events
- Ant Design layout, cards, buttons, date picker, and grid
- SPA navigation with `Home`, `Dashboard`, `About`, `Register`, and `Login`
- Dynamic article detail page at `/a/:aid`
- Static article JSON data for the landing page
